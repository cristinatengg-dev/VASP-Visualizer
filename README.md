# SCI Visualizer (scivisualizer.com)

SCI Visualizer 是一个面向“原子结构与材料模拟结果”的网页可视化工具：把科研/工程里常见的结构文件与计算输出，变成浏览器里可交互的 3D 画面，并支持测量与导出，方便沟通、检查与展示。

你可以把它理解成：

- 原子世界的 3D 查看器（像看 3D 地图一样看原子怎么排布）
- 结果文件阅读器（把复杂的模拟输出文件变成直观图形）
- 带自然语言入口的助手工作台（帮你搭结构、做渲染图等）

## 这个网站用来做什么？（零基础版）

很多材料/化学/能源相关研究会在计算机里模拟材料的原子结构与性质（例如 VASP 这类第一性原理计算、分子动力学等）。这些结果通常是一堆结构文件或大体量数据文件，传统上需要安装专业软件才能查看。

SCI Visualizer 的目标是：

- 让你在浏览器里直接打开、旋转、缩放查看结构
- 用可视化方式快速回答“结构长什么样”“关键原子间距离/角度是多少”“电子云大概分布在哪里”
- 一键导出图片/视频用于报告、论文或团队沟通

## 不懂这个领域也能理解的几个概念

- 原子结构：很多“小球”按规律排布，每个小球代表一种元素（例如 C=碳、O=氧）。
- 键/距离：两原子是否“连接”常用“距离是否足够近”来判断；也可以直接测量两原子间距离（常用单位 Å，埃，约等于 0.1 纳米）。
- 晶胞/周期性边界（PBC）：晶体可以看作无限重复；软件常用一个“最小重复单元”代表整个材料。
- 电子云/体数据：有些计算会输出三维网格数据（电子密度/电势等），常见可视化方式是“等值面”（类似 3D 等高面）。
- 轨迹：分子动力学会产生很多帧结构，连起来就是动画。

## 主要功能（从使用者角度）

### 结构 3D 可视化与交互

- 旋转/缩放/平移查看原子结构
- 选中/框选原子，测量两原子距离、三原子夹角
- 显示/隐藏晶胞边框，切换不同渲染风格（球棍/线框/经典风格等）

### 结果文件解析与可视化

- 结构文件（例如 POSCAR/CONTCAR/CIF）：显示原子与晶格
- 体数据（例如 CHGCAR/LOCPOT 等）：生成等值面/体渲染（电子云/电势分布）
- 轨迹文件（例如 XDATCAR）：播放结构随时间变化的动画

网站会尽量在浏览器本地快速解析；遇到大文件或复杂格式时，会调用后端服务进行更稳健的解析。

### 导出与分享

- 导出截图（高清）
- 导出批量图片或视频（取决于站点功能与配额策略）

### Agent / 助手工作台

- 建模助手（Modeling）：用自然语言描述“搭一个 Cu(111) 表面”“在表面放一个分子”等，系统将其解析为结构操作并生成结构。
- 渲染助手（Rendering）：把科学描述/论文内容/风格偏好编译成更适合生成封面/海报的输入，并调用图像生成能力输出候选图。
- 计算助手（Compute）：把意图转成计算输入集，并可对接集群提交流程（具体取决于部署环境是否配置）。

## 它不是什么（边界）

- 它不是计算引擎本身：真正的数值计算通常仍需要在本地或计算集群上完成。
- 它不追求替代所有专业后处理软件：目标是“快速理解 + 交互检查 + 可视化输出”。

## 一个最小使用流程（零基础版）

1. 打开网站进入应用页面
2. 上传一个结构文件（例如 CONTCAR 或 POSCAR）
3. 在 3D 视图中旋转/缩放查看结构
4. 选择两个原子查看距离（或启用成键规则显示连接关系）
5. 如有体数据文件（例如 CHGCAR/LOCPOT），上传后查看等值面
6. 导出一张图用于报告/分享
